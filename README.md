# Calculator Sandbox: Infinite 🔢🎮

欢迎来到**计算器沙盒：无限**！这是一个结合了数学运算、RPG 成长和 Roguelike 生存的命令行游戏。在这个无限生成的数字矩阵中，你将扮演一个数字节点，通过精妙的计算和战斗来对抗系统错误。

## 🌟 核心特性

- **无限矩阵**: 基于种子生成的无限随机地图。
- **数学驱动**: 所有的道具都是数学运算符（`+`, `-`, `*`, `/`, `^`, `%`），直接操纵你的血量或分数。
- **RPG 成长**: 独立的 XP 系统，通过收集数字和击杀敌人升级，解锁生命、体力、伤害和范围的强化。
- **策略道具**: 使用 `~` (翻转)、`{` (升序)、`}` (降序) 来重组你的分数，玩转数字逻辑。
- **史诗 BOSS**: 面对巨型多格实体 **Math Lord**，躲避它的八向弹幕。
- **双重模式**:
  - **挑战模式**: 180秒内获得 200 分并寻找每 15 秒位移一次的动态传送门。
  - **无尽模式**: 在不断增强的敌人压力下挑战生存极限。

## 🕹️ 快捷键指南

| 按键 | 功能 |
| :--- | :--- |
| **W/A/S/D** | 移动 / 菜单导航 |
| **Space** | 攻击 (消耗体力) |
| **E** | **AC (All Clear)** 终极技能 (清屏) |
| **P** | 暂停 / 恢复 |
| **Q** | 退出 / 返回主菜单 |

## 🛠️ 安装与运行

1. 确保已安装 Python 3.8+。
2. 安装依赖库：

   ```bash
   pip install rich
   ```

3. 启动游戏：

   ```bash
   python main.py
   ```

## 📜 开发者备注

本项目是一个展示 Python `rich` 库强大渲染能力的实验性沙盒游戏。所有的逻辑（包括碰撞、多格实体、投射物）均在轻量级的命令行环境中实现。
